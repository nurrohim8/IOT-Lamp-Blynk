Program ini untuk praktikum kendali lampu dengan mikrokontroler ESP32 dan Blynk.

Alat dan bahan:
- ESP32 = 1 pcs
- Relay 5V = 1 pcs
- Lampu = 1 pcs
- Kabel = secukupnya
- HP terinstal Blynk

Berikut ini gambar rangkaiannya.
