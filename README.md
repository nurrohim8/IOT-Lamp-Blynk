Program ini untuk praktikum kendali 2 buah lampu dengan mikrokontroler ESP32 dan Blynk.

Alat dan bahan:
- ESP32 = 1 pcs
- Relay 5V = 2 pcs
- Lampu = 2 pcs
- Kabel = secukupnya
- HP terinstal Blynk
